# YOJtech Business Website

[view link](https://profound-mousse-a8623d.netlify.app/)


This is a responsive business website template built with HTML, CSS, and JavaScript. The site showcases services, portfolio, team, and contact information for YOJtech.

## Features

- Modern, responsive design
- Home, About, Services, Portfolio, Team, and Contact pages
- Animated sections using AOS (Animate On Scroll)
- Newsletter subscription form
- Contact form (requires backend for full functionality)
- Integrated with Bootstrap, Bootstrap Icons, and other vendor libraries

## Getting Started

1. **Clone or Download** this repository.
2. **Open `index.html`** in your browser to view the homepage.
3. **Customize content** in the HTML files as needed.
4. **Deploy** to your preferred static hosting service (e.g., Netlify, Vercel).
